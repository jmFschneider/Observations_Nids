Aide
Quand utiliser cet outil ?
  • Anciennes communes : Communes fusionnées depuis 2010 (ex: "Les Praz" → "Chamonix-Mont-Blanc")
  • Communes étrangères : Observations en Suisse, Belgique, etc.
  • Lieux-dits : Hameaux ou lieux-dits non référencés comme commune officielle
  • Erreurs OCR récurrentes : Variantes orthographiques détectées par l'OCR
Exemples de recherches :
  • Les Praz + Haute-Savoie → Lieu-dit de Chamonix
  • Genève + Suisse → Ville suisse
  • Blonville-sur-Mer → Commune du Calvados
Limitations :
  • Limite de 1 requête par seconde sur Nominatim (respectez les règles d'usage)
  • Les coordonnées sont approximatives (centre de la commune, ~5 km de précision)
  • Le code INSEE et le code postal devront être complétés manuellement après ajout
